githubのリポジトリを別端末で共有するには

  1. githubへアクセスして、リポジトリの管理画面へ。そしてcodeをクリックしリポジトリのリンクをコピーする。

  2. 保存したいローカルの場所へ行き、右クリックでGit Bash hereを選択する。

  3. 次のコマンドを入力
    4. git clone

用語まとめ

  • Git(ギット)
    • Gitとは分散型バージョン管理システムの一種である。 集中管理用のサーバーを持たないため、サーバーが機能せずともバージョン管理を行うことが可能である。

  • リポジトリ(Repository)
    • githubなどのようなバージョン管理システムにおける、 ソースコードやディレクトリ構造のメタデータを格納するデータ構造のこと。 wikipediaより引用>

  • コミット(commit)とプッシュ(push)
    Gitでローカルのリポジトリに変更を加える操作をコミットといい、 その変更をローカルからリモートに変更する操作をプッシュという。
